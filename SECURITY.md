# Security

## Responsible Disclosure

If you identify an issue in this repository, please report it responsibly to `peter.cagbenyega@gmail.com`.

## Scope

- This repository is demonstration and reference-oriented.
- It is intended to document architecture patterns and public technical work.
- No real secrets should be stored in this repository.
- Example configurations should be treated as reference material, not direct deployment instructions.

## Handling Notes

- Do not submit credentials, tokens, private keys, or sensitive environment data.
- Security-related observations should include enough context to reproduce the issue safely.
