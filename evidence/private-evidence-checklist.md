# Private Evidence Checklist

This checklist is for private attorney review only. It does not contain private documents.

## Identity Documentation
- Passport biographic page
- Lawful status records

## Professional Evidence
- Resume/CV
- AWS certifications
- Employment verification letters
- Reference letters
- Technical recommendation letters

## Education Evidence
- UMGC enrollment evidence
- Degree plan
- WES evaluation
- Bachelor's degree documents

## Technical Portfolio Evidence
- Public GitHub repositories
- Architecture diagrams
- README files
- Evidence packs
- Public portfolio screenshots
- Project summaries

## Community and Professional Responsibility Evidence
- Human-services employment verification
- Training certificates, if appropriate
- Supervisor or colleague letter
- Documentation/compliance-related training records, if appropriate

## Business/Organizational Evidence
- Cloud Nexus Hub LLC formation documents
- EIN letter
- Business plan
- Public-interest technical roadmap
- Non-commercial/public documentation statement

## Important Privacy Rule
Do not place private identity documents, business tax records, employment authorization records, or protected personal information inside the public GitHub repository.
